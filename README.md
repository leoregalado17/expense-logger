# Expense Logger PWA

A mobile-first Progressive Web App for logging daily expenses. Built as a single HTML file with no dependencies.

## Features

- Dark mode UI optimized for mobile
- Add expenses by category with quick entry
- Installable as a PWA (add to home screen)
- Works offline — all data stored locally

## Usage

Open the live app: [https://leoregalado17.github.io/expense-logger/expense_logger.html](https://leoregalado17.github.io/expense-logger/expense_logger.html)

To install on your phone:
1. Open the URL in Safari (iOS) or Chrome (Android)
2. Tap **Share** → **Add to Home Screen**
3. The app will launch fullscreen like a native app
